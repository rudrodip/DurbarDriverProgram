Command code:
    Motor control command: m[DIR][MOTOR][PWM]
    DIR -> f - forward, b - backward, r - right, l - left, s - stop, c - custom
    Motor -> 0 or 1
    PWM -> (0-255)


    Servo Control Command: s[servo_number][pos]
    pos -> (0-180)

    Arduino Serial Code:
        t[temp]h[humid]s[smoke]
        d[distance]
